HM Plot

Utilize Deep learning in assiting the concept cluster stage of the design process




Detailed Explanation

Use the natural language processing algorithm (Word2Vec) to create machine clusters through analyzing the description inputs of students submittion



Create plots mapping the different number of machine clusters to human clusters to explore patterns that can shed insights in human clustering



A pattern where the algorithm breaks apart a large human cluster into is called under clusting. A pattern where algorithm combines several human clusters into a large machine cluster is called over clustering.


(This a continuous research project initially carried out by Chengwei Zhang (QsingHua PHD Researcher)
